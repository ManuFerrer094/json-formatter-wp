# JSON Formatter Shortcode for WordPress

This WordPress plugin provides a user-friendly interface to format and validate JSON data. Users can simply use the shortcode `[jsonformatter]` to integrate the JSON formatter tool into their WordPress content.

## How to Use

1. Add the shortcode `[jsonformatter]` to any post or page content.
2. Drag and drop your JSON data or enter it manually.
3. Choose formatting options and click "Process JSON."
4. Download, copy to clipboard, or clear the formatted JSON result.
